1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

  • CSS digunakan untuk mengatur tampilan halaman web agar lebih menarik dan konsisten.
  • JavaScript digunakan untuk menambahkan interaksi dan dinamika pada halaman web.

2. Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

CSS dan JavaScript dapat diintegrasikan melalui:

  • Inline: langsung di dalam elemen HTML.
  • Internal: menggunakan tag <style> dan <script> di file HTML.
  • Eksternal: menghubungkan file .css atau .js terpisah menggunakan <link> dan <script src="">.

3. Jelaskan apa saja keuntungan menggunakan JavaScript dan CSS?

Keuntungan CSS:
  • Desain lebih konsisten dan mudah diubah.
  • Responsive design lebih mudah dibuat.
Keuntungan JavaScript:
  • Membuat halaman lebih interaktif.
  • Dapat memproses data tanpa reload halaman.

4. Jelaskan apa saja keuntungan menggunakan Bootstrap?

  • Mempermudah pembuatan desain responsif.
  • Tersedia banyak komponen siap pakai.
  • Desain jadi konsisten tanpa banyak coding.

5. Contoh Penggunaan Bootstrap

Contoh kode:

<div class="alert alert-success" role="alert">
  Ini adalah alert sukses dari Bootstrap!
</div>

Penjelasan:

  • Kode ini menampilkan alert warna hijau yang menandakan pesan sukses.
  • Class alert alert-success adalah bawaan dari Bootstrap.
